%=====================================================================================
\section{Bachelor's Thesis}\label{section:BT}
%=====================================================================================

The source code files\footnote{\url{https://github.com/europ/VUTBR-FIT-BT}} of this bachelor's thesis are located inside of \uv{\textit{Bachelor's Thesis}} directory which includes the following subdirectories:

\begin{itemize}

    \item \uv{\textit{PDF}} -- PDF version of thesis, presentation and assignment.
    \item \uv{\textit{presentation}} -- presentation source code files.
    \item \uv{\textit{thesis}} -- thesis source code files.

\end{itemize}

%=====================================================================================
\section{Bachelor's Thesis - Implementation}\label{section:BT-I}
%=====================================================================================

The \uv{\textit{Bachelor's Thesis - Implementation}} directory includes the implementation source codes. This directory is a Git directory which contains the ManageIQ Bot's source codes along with the branches which includes the source codes related to the pull requests which are mentioned in footnotes of Chapter \ref{chapter:implementation}.

%=====================================================================================
\section{Bachelor's Thesis - Setup Guide}\label{section:BT-SG}
%=====================================================================================

The setup guide\footnote{\url{https://github.com/europ/VUTBR-FIT-BT-IMPL}} for the bot is written in a markdown format and it is located inside of \uv{\textit{Bachelor's Thesis - Setup Guide}} directory. This directory also includes a PDF and \LaTeX\ version of these markdown files located in subdirectories named by the version type. The starting point of the setup guide can be found in the main file of the guide - \uv{\textit{README}} file.
